📱 3. Android 手机App

Android 系统最初就是用 Java 写的！
虽然现在 Kotlin 比较流行，但 Java 仍然是 Android 开发的重要语言。

☁️ 4. 企业系统与大数据

Java 在 大型企业系统和服务器 上非常强势。

💻 5. 游戏开发

虽然不像 C++ 那样主流，但 Java 也能做游戏。

著名的 《Minecraft 我的世界》，核心就是用 Java 写的！

也可以用 LibGDX、jMonkeyEngine 等引擎制作 2D/3D 游戏。

🔒 6. 工具与嵌入式系统

Java 还能写各种小工具、命令行工具、甚至运行在嵌入式设备上（例如刷卡机、智能电视）。
